# ⭐ To-Do List em Python ⭐

Projeto pessoal desenvolvido em **Python** com foco em praticar lógica de programação, organização de código e manipulação de arquivos.

Aplicação simples que roda no **terminal** e permite gerenciar tarefas do dia a dia.

---

## ⭐ Funcionalidades

* Adicionar tarefas
* Listar tarefas
* Marcar tarefas como concluídas
* Remover tarefas
* Salvar tarefas em arquivo
* Carregar tarefas ao iniciar o programa

---

## ⭐ Estrutura de uma tarefa

Cada tarefa possui:

* Descrição
* Status (pendente ou concluída)
* Grau de urgência

---

## ⭐ Estrutura do projeto

```
todo-list/
├── main.py
├── data/
│   └── tasks.json
└── README.md
```

---

## ⭐ Como executar

* Ter Python instalado (versão 3+)
* Executar no terminal:

```bash
python main.py
```
